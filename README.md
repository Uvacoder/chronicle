## Running Locally

This is a [Next.js](https://nextjs.org/).

Run the development server with

```bash
yarn dev
```

Open [localhost:3000](http://localhost:3000) with your browser to see the result.

## Building for Production

Create a build

```bash
yarn build
```

Run the server

```bash
yarn start
```
